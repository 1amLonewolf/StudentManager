# Certificate Template Placeholder

## How to Add Your Certificate Template

1. **Create or obtain your certificate design**
   - Recommended size: A4 landscape (3508 x 2480 pixels at 300 DPI)
   - Format: PNG with transparent text areas (or solid background)

2. **Save as template.png**
   - Place this file at: `static/certificates/template.png`

3. **The system will automatically:**
   - Overlay student name
   - Add course name ("Computer Packages")
   - Add completion date
   - Add modules completed
   - Add grade (Pass/Credit/Distinction)
   - Add certificate number

## Text Placement (Default)

The system places text at these positions:
- **Student Name**: Center, large font (36pt)
- **Course Name**: Below name (18pt)
- **Completion Date**: Below course name (14pt)
- **Modules**: Below date (12pt)
- **Grade**: Below modules (16pt)
- **Certificate Number**: Bottom center (10pt)

## Tips

- Leave empty space in the center for student name overlay
- Use high contrast colors for readability
- Test print one certificate before bulk generation
- Keep file size under 5MB for faster processing

## Alternative: Use Built-in Design

If no template is provided, the system generates a basic certificate with:
- Blue and gold borders
- "CERTIFICATE OF COMPLETION" header
- Decorative corner elements
- Professional layout
